## Wat kunt u doen?

De Okasie Partner API ondersteunt:

- **Advertenties ophalen** - Bekijk alle advertenties binnen uw toegangsbereik
- **Advertenties aanmaken/bijwerken** - Synchroniseer uw inventaris met Okasie
- **Bulk operaties** - Upsert tot 100 advertenties per aanvraag
- **Locatiebeheer** - Beheer meerdere dealerlocaties

## API Versioning

De huidige API-versie is **v1**. Alle endpoints beginnen met:

```
https://www.okasie.be/api/external/v1/
```

<Note>
  API-toegang vereist een partneraccount. Neem contact op met [info@okasie.be](mailto:info@okasie.be) om toegang aan te vragen.
</Note>
