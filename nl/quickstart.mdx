# Snel aan de slag

Deze handleiding helpt u bij uw eerste API-aanroep naar de Okasie Partner API.

## Vereisten

<Steps>
  <Step title="Vraag API-toegang aan">
    Neem contact op met [info@okasie.be](mailto:info@okasie.be) om partneraccount aan te vragen.
    U ontvangt:
    - API-sleutel (geheim)
    - Partner-ID
    - Toegangsbereik (welke dealers u kunt benaderen)
  </Step>

  <Step title="Bewaar uw inloggegevens">
    Sla uw API-sleutel veilig op. Deel deze nooit in clientcode of openbare repositories.
  </Step>
</Steps>

## Uw eerste API-aanroep

### Advertenties ophalen

<CodeGroup>
```bash cURL
curl -X GET "https://www.okasie.be/api/external/v1/listings?status=active&pageSize=10" \
  -H "Authorization: Bearer UW_API_GEHEIM"
```

```javascript Node.js
const response = await fetch(
  "https://www.okasie.be/api/external/v1/listings?status=active&pageSize=10",
  {
    headers: {
      Authorization: `Bearer ${process.env.PARTNER_SECRET}`,
    },
  }
);

const data = await response.json();
console.log(`Gevonden: ${data.pagination.totalItems} advertenties`);
```

```python Python
import os
import requests

url = "https://www.okasie.be/api/external/v1/listings"
headers = {"Authorization": f"Bearer {os.getenv('PARTNER_SECRET')}"}
params = {"status": "active", "pageSize": 10}

response = requests.get(url, headers=headers, params=params)
data = response.json()
print(f"Gevonden: {data['pagination']['totalItems']} advertenties")
```
</CodeGroup>

## Veelvoorkomende problemen

<AccordionGroup>
  <Accordion title="401 Unauthorized">
    Controleer of uw API-sleutel correct is en de `Authorization: Bearer` prefix bevat.
  </Accordion>

  <Accordion title="403 Forbidden">
    Uw API-sleutel heeft geen toegang tot de aangevraagde resource. Controleer uw toegangsbereik.
  </Accordion>
